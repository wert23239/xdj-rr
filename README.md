# XDJ-RR Web DJ Controller

A full-featured, browser-based DJ controller inspired by the Pioneer XDJ-RR. Built with vanilla JavaScript and the Web Audio API — no frameworks, no dependencies on the client side.

![XDJ-RR Screenshot](screenshots/default.png)

## Features

### Dual Decks
- **Audio playback** with Web Audio API (supports MP3, WAV, AAC, M4A, OGG, FLAC, AIF)
- **Jog wheels** with touch/mouse scratch and nudge
- **Tempo sliders** (±8% range, 0.1% precision)
- **Hot cues** (4 per deck, right-click to delete)
- **Beat-synced loops** (1/2, 1, 2, 4, 8 beats)
- **Beat jump** (±1, ±8, ±32 beats)
- **Vinyl brake** mode with spindown effect
- **Reverse playback** (hold REV button)
- **Slip mode** — scratching/loops return to original position
- **Quantize** — snaps hot cues and loops to beat grid
- **BPM detection** and manual **TAP tempo**
- **Key detection** with Camelot wheel display

### Mixer
- **3-band EQ** (Hi/Mid/Lo) with kill switches per channel
- **Gain knobs** with auto-gain normalization
- **Channel volume faders** and **VU meters**
- **Crossfader** with 3 curve modes (Smooth, Sharp, Power)
- **Auto-crossfade** buttons (animated transition)
- **Color FX filter** knobs per channel
- **Noise generator** toggle
- **CUE/Master mix** knob for headphone monitoring
- **Master volume** with limiter indicator
- **Harmonic mixing indicator** (Camelot wheel compatibility)

### Beat FX (per channel)
- Echo, Reverb, Delay, Flanger, Phaser, Bitcrusher

### Sampler
- 4 synthesized pads: Air Horn, Siren, Scratch, Crowd Cheer

### Waveforms
- **Overview waveforms** (full track, click to seek)
- **Zoomed waveforms** with beat grid overlay (2×–16× zoom)
- **Parallel waveform** view for visual beat matching
- **3 color modes**: Frequency spectrum, Solid, Grayscale
- **Hot cue markers** and **loop region** overlay

### Auto-Mix Transitions
- **16-Bar Blend** — smooth 16-bar crossfade with EQ swap
- **Echo Out** — echo trail into next track
- **Cut Swap** — instant hard cut

### Track Browser
- Search/filter tracks in real time
- Sort by date, name, or file size
- Drag-and-drop tracks onto decks
- Load buttons (D1/D2) on hover

### Song Discovery
- **YouTube + SoundCloud search** via yt-dlp
- **One-click download** to DJ library (converts to WAV)
- Download progress tracking

### Tracklist Recording
- **REC** button starts a named session
- Auto-logs tracks on play, detects crossfader transitions
- Manual log buttons per deck
- Export tracklist as text file
- Session history stored in Supabase

### Playlists
- Create, load, and delete playlists (Supabase-backed)
- Add tracks from either deck

### Recording
- **REC MIX** — records the master output as WAV using MediaRecorder

### Display Modes
- **Performance mode** — hides browser, maximizes deck area
- **Mini-mixer mode** — compact mixer layout
- **Light/Dark theme** toggle

### Mobile / PWA
- **Mobile-first layout** with deck switching (swipe or tap D1/D2)
- Slide-up mixer sheet, full-screen browser overlay
- **PWA installable** (manifest.json + service worker)
- Offline caching for app shell

## How to Run

### Prerequisites
- **Node.js** ≥ 18
- **yt-dlp** (for song discovery/download): `brew install yt-dlp`
- Audio files in `~/Music/DJ/`

### Install & Start

```bash
cd xdj
npm install
npm start
```

Open **http://localhost:3000** in your browser.

### Run Tests

```bash
npx playwright install chromium
npx playwright test
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Q` | Play/Pause Deck 1 |
| `W` | Cue Deck 1 |
| `P` | Play/Pause Deck 2 |
| `O` | Cue Deck 2 |
| `E` | Sync Deck 1 |
| `I` | Sync Deck 2 |
| `1`–`4` | Hot Cues Deck 1 |
| `7`–`0` | Hot Cues Deck 2 |
| `Z` / `X` | Nudge Deck 1 −/+ |
| `,` / `.` | Nudge Deck 2 −/+ |
| `B` | Toggle Browser |
| `?` | Show/Hide Shortcuts |
| `Space` | Auto-crossfade toggle |

## Tech Stack

- **Frontend**: Vanilla JS, Web Audio API, Canvas API
- **Backend**: Node.js + Express
- **Database**: Supabase (tracklists, playlists)
- **Search/Download**: yt-dlp
- **Testing**: Playwright
